# Plan — 修复

## 1. Day 小字日期仅在今天显示

**问题**：当前任何日期下都显示小字日期+星期，但只有"今天"需要额外标注具体日期。其他日子标题本身已经是具体日期了，小字就重复了。

**修改**：`day-date-label` 只在今天时显示（如 "2026年5月5日 周一"），其他日子隐藏。

## 2. 右上角快捷跳转按钮修复

**问题**：四个视图右上角的日历图标点击没反应。`input.showPicker()` 兼容性差。

**修改**：改用自定义弹窗方案

- **Day**：点击弹出日期选择（年-月-日），确认后跳转到那天
- **Week**：点击弹出日期选择（年-月-日），确认后跳转到包含那天的周
- **Month**：点击弹出月份选择（年-月），确认后跳转到那个月
- **Year**：点击弹出年份输入，确认后跳转到那年

弹窗方式：用 `prompt` 或自定义小型 modal，包含简洁的输入控件。

## 3. 去掉 Week/Month/Year 标签旁的多余日期文字

**问题**：Week 的 week-label、Month 的 month-label 已经显示了日期范围，不需要额外再显示。

**修改**：确认没有多余的日期文字叠加在标签旁边。

## 4. Week/Month/Year 左右滑动切换未生效

**问题**：之前说去掉箭头纯靠滑动，但滑动切换没有实际生效。

**修改**：确保 Week 左右滑、Month 上下滑、Year 左右滑的触摸事件正确绑定并能切换视图。
